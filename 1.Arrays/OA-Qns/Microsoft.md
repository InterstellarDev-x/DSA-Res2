## OA Overview

Microsoft OA (2024–2026) is hosted on HackerRank or Codility. It typically includes 2–3 coding problems in 60–90 minutes. Array problems often test fundamental manipulations and medium-difficulty algorithms. Microsoft tends to prefer problems that can be solved correctly over those requiring advanced data structures.
