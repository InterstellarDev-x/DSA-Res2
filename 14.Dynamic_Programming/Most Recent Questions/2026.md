## 2026 trend analysis (through June)

- **The universal-staple core is rock solid:** Coin Change and House Robber remain the must-prep OA pair at Amazon, Microsoft, and Goldman Sachs — the same conclusion as [2024](./2024.md) and [2025](./2025.md).
- **Word Break has fully arrived at Amazon** — no longer "rising", it is now a default OA item alongside Coin Change.
- **Edit Distance is entrenched as the Google DP signature** and continues as the Microsoft onsite mainstay with the production diff-tool framing.
- **Interval DP (Burst Balloons) remains the Google L5 hard differentiator** — still the cleanest separator between L4 and L5 candidates.
- **Stock problems are consistent across all companies** and increasingly tested at the fee/cooldown end at Goldman Sachs, exercising the unified state machine.
- **LIS O(n log n) is now the baseline expectation** at Google for 2025-2026; presenting only O(n²) signals an incomplete answer in senior loops.
